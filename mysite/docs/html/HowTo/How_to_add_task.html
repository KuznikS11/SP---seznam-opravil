Howto add a new task¶

  1. login; if you don’t have an account yet, first register
  2. click Add task on your left side or click Add new task on your right side
  3. Fill the form you see with informations about the task you want to add
  1. Click Add new task to add you task
  2. Click Cancel if you changed your mind and you don’t want to add new task
